# Service Soins Infirmiers

## À quoi ça sert ?

C'est le service qui gère les **soins infirmiers** administrés aux patients hospitalisés.

## Ce qu'il fait

- **Créer un plan de soins** : le médecin ou l'infirmier définit les soins à administrer (type, fréquence, instructions)
- **Suivre l'administration** : l'infirmier enregistre chaque soin effectué
- **Gérer les non-administrations** : documenter pourquoi un soin n'a pas pu être fait (refus du patient, contre-indication...)
- **Traçabilité** : qui a fait quoi, quand

## Exemple concret

Mme Traoré est hospitalisée. Le plan de soins :
1. **Pansement** : changer le pansement 2 fois par jour
2. **Injection antibiotique** : Ceftriaxone 1g, 1 fois par jour à 8h
3. **Contrôle tension** : TA 3 fois par jour

L'infirmière de garde :
- 8h00 : injection faite, noté
- 8h15 : TA 12/8, noté
- 10h00 : pansement changé, noté
- 18h00 : TA 11/7, noté
- 20h00 : pansement : patient refuse (dort) → noté avec motif

## Pourquoi c'est important ?

- Le médecin peut vérifier que les soins prescrits ont bien été administrés
- En cas de changement d'équipe, le nouvel infirmier sait ce qui a été fait
- Traçabilité complète pour la qualité des soins
